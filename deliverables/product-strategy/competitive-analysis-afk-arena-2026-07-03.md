## 📌 TL;DR（执行摘要）

- **核心目标**：为新手开发者仿制《剑与远征》核心玩法 MVP（放置战斗 + 英雄收集养成 + 关卡推进，移动端）提供全面竞品分析和技术选型支撑
- **关键决策**：① 技术选型推荐 **Unity**（市占率 51%，放置 RPG 适配度最高，生态最丰富）；备选 Godot 4（纯学习/零预算）；② MVP 聚焦 6 个核心系统（离线收益、自动战斗+大招、3-4 阵营克制、英雄升品质、关卡推进、基础抽卡+保底），控制规模在 10-20 英雄 / 50-100 关；③ 题材差异化 + 买断制/纯装饰内购是独立开发者可行路径

## 1. 竞品全景图（来自竞析）

### 1.1 品类定位

放置 RPG 是全球手游市场中增长最强劲的 RPG 细分赛道：
- 占 RPG 总收入约 **15%**，近五年收入持续上升
- 下载量位居 RPG 各细分品类**第一**（占近 20%），轻量化玩法是核心吸引力
- "离线也能变强"的核心理念精准命中碎片化时间需求

## 4. 用户感知（来自瑞思）

### 4.1 玩家核心满意度因素

| 满意度因素 | 机制体现 | 玩家感知 |
|-----------|---------|---------|
| **碎片化时间适配** | 每日约 10 分钟完成日常，12h 挂机上限 | "备胎手游"定位精准 |
| **共享等级（共鸣水晶）** | 只需培养 5 个英雄 | 大幅减负，释放阵容策略空间 |
| **收集养成成就感** | 100+ 英雄、30 抽保底、心愿单 | 抽卡门槛不高，保证目标感 |
| **策略深度** | 站位、种族克制、RE 大法 | "去掉放置元素依然优秀的卡牌游戏" |
| **渐进式付费** | 付费系统一周后才开放 | 早期逼氪氛围不浓 |

### 4.2 玩家核心痛点（按严重程度排序）

| 痛点 | 严重度 | 玩家原话 |
|------|--------|---------|
| **数值膨胀与 Power Creep** | ⭐⭐⭐⭐⭐ | "多花 1000 块和别人花 100 块差不了多少" |
| **付费取胜严重化** | ⭐⭐⭐⭐⭐ | "小微氪都别想玩舒服，要么零氪收菜要么爆米" |
| **恶性循环已现** | ⭐⭐⭐⭐⭐ | "玩家流失→增氪金→玩家反馈恶劣→玩家流失" |
| **系统过载（60+ 系统）** | ⭐⭐⭐⭐ | 认知过载，攻略需超长篇幅才能覆盖 |
| **内容重复与枯竭** | ⭐⭐⭐⭐ | "电子表格模拟器""上班感" |
| **进度墙** | ⭐⭐⭐⭐ | 200 级粉尘墙、300 级经验墙 |

### 4.3 流失节点

| 流失节点 | 阶段 | 原因 |
|---------|------|------|
| 早期卡关 | 4-4 后 | 不理解心愿单/升阶策略，资源浪费 |
| 粉尘墙 | ~200 级 | 粉尘不足，升级停滞 |
| 经验墙 | ~300 级 | 经验不足，进一步停滞 |
| PvP 天花板 | 终局 | P2W 严重，免费玩家无法竞争 |
| 内容枯竭 | 后期 | 日常重复、数值膨胀 |

### 4.4 玩家核心需求优先级

| 优先级 | 需求 | 依据 |
|--------|------|------|
| **P0** | 碎片化时间利用（每日 10-15 分钟） | 社区高频讨论 × 极高影响度 |
| **P0** | 成长反馈与成就感（持续正反馈） | 留存行为核心驱动 |
| **P1** | 低操作门槛（自动战斗、离线） | "备胎手游"选择动机 |
| **P1** | 公平感与付费平衡 | 最大抱怨集中点 |
| **P2** | 策略深度（阵容/站位/克制） | 豆瓣好评核心 |
| **P2** | 弱社交（不强制实时交互） | "半单机"状态 |
| **P3** | 美术品质与世界观 | 独立游戏差异化机会 |
| **P3** | 内容新鲜度（赛季制） | AFK Journey 验证有效 |

### 4.5 独立开发者用户接受度

基于 GDC 2023 独立游戏玩家研究（4800+ 受访者）：
- **40%** 受访玩家在过去 12 个月内购买过至少一款独立游戏
- 独立游戏买家用"独特""实验性""创新"描述独立游戏
- 玩家追求"与 3A 不同的体验——更新颖、更有创意、更敢冒险"
- **2D 完全可接受**，但要求风格统一
- **音乐品质是差异化机会**："背景音乐和配乐是重头戏"
- **买断制（$10-15）比 F2P+内购更受欢迎**，"微交易影响游戏性时玩家会反感"

## 5. SWOT 分析

### 5.1 仿制 AFK Arena 项目 SWOT

**优势（Strengths）：**
- 放置 RPG 玩法模式成熟，有明确标杆可参考
- 核心循环简洁（挂机→升级→推关→抽卡），MVP 范围可控
- 2D 技术门槛相对低，适合新手开发者
- 离线收益机制对服务器压力小（异步计算）

**劣势（Weaknesses）：**
- 新手开发者缺乏数值平衡经验（AFK 的数值设计极其精细）
- 美术资源是核心竞争力，新手难以达到商业品质
- 商业化系统复杂度高，MVP 难以实现
- 缺乏运营经验

**机会（Opportunities）：**
- 放置 RPG 是 RPG 增长最快的细分赛道（CAGR 9.8-12.5%）
- 3A 大厂较少涉足轻量放置，独立开发者有空间
- 小程序/H5 平台提供新渠道
- AI 辅助开发（美术生成、数值平衡）降低门槛

**威胁（Threats）：**
- 莉莉丝等大厂已占据头部（AFK Arena + AFK Journey 双产品线）
- 品类同质化严重，用户审美疲劳
- 获客成本持续上升，小团队获客困难
- 苹果/谷歌对抽卡机制监管趋严

### 5.2 对标竞品 SWOT（从仿制者视角）

| 竞品 | 可借鉴的优势 | 可利用的弱点 |
|------|-------------|-------------|
| **AFK Arena** | 离线收益设计、共鸣水晶、关卡教育设计、精细化商业化 | 2D 画面老化、中期倦怠、社交极弱、内容更新慢 |
| **AFK Journey** | 3D 大世界、赛季重置、共享等级/装备、UX 优秀 | 肝度增加、包体大、设备要求高、赛季重置可能劝退 |
| **放置奇兵** | 6 阵营深度克制、8 年长线运营、VIP 体系 | 画面老旧、新手门槛高、系统繁杂、创新不足 |
| **不休的乌拉拉** | 组队放置创新、社交化设计 | 玩法偏小众、留存难度大 |

## 6. 差异化机会

### 6.1 不可模仿的领域（大厂壁垒）
- ❌ 精细化动态推荐商业化系统（需要大量数据积累和算法能力）
- ❌ 顶级 2D/3D 美术品质（需要专业美术团队）
- ❌ 大规模买量获客（需要充足资金）

### 6.2 可行的差异化方向

| 方向 | 推荐度 | 说明 |
|------|--------|------|
| **题材差异化** | ⭐⭐⭐⭐⭐ | 同样系统框架换皮（科幻/修仙/末日/萌宠），低成本吸引不同用户群 |
| **MVP 极简化** | ⭐⭐⭐⭐⭐ | 砍掉非核心系统，只保留 6 大核心，先验证循环再扩展 |
| **平台差异化** | ⭐⭐⭐⭐ | 优先做微信小游戏/H5，利用社交裂变获客，降低 CPI |
| **玩法融合创新** | ⭐⭐⭐⭐ | 放置+Roguelike 深度融合、放置+自走棋、放置+塔防 |
| **买断制/纯装饰内购** | ⭐⭐⭐⭐ | 独立游戏买家更接受买断制，避免 P2W 恶名 |
| **优质原创音乐** | ⭐⭐⭐ | 低成本高回报的差异化点，独立游戏玩家重视音乐 |
| **轻社交** | ⭐⭐⭐ | 好友助战、排行榜竞争、公会协作（不需实时） |

### 6.3 MVP 最小系统集建议

**必须做（P0）：**
1. ✅ 离线收益系统（核心卖点，12h 上限）
2. ✅ 自动战斗 + 手动大招（核心玩法）
3. ✅ 3-4 个阵营 + 循环克制关系
4. ✅ 英雄升级 + 升品质（核心养成线）
5. ✅ 关卡推进 + 卡关设计（目标感）
6. ✅ 基础抽卡 + 30 抽保底（获取系统）

**强烈建议（P1）：**
7. ⚠️ 共鸣水晶/共享等级（养成减负关键）
8. ⚠️ 限时礼包（最简单有效的商业化）

**暂缓（P2+）：**
- ❌ 动态推荐系统、Roguelike 关卡、专属武器/神器、PVP 竞技场、公会系统

### 6.4 核心玩法循环（AFK Arena 标杆拆解）

```
挂机产出资源（离线/在线）
    ↓
英雄升级 → 属性提升（指数函数，前期增幅大）
    ↓
推关验证 → 卡关 → 升级/升品质/调阵容
    ↓
任务奖励 → 钻石 → 抽卡 → 品质提升
    ↓
品质固定值带来显著反馈 → 强化对钻石的需求
    ↓
关注任务目标（推关数/爬塔数）→ 回到挂机
```

**关键数值设计（AFK Arena 标杆）：**
- 离线粉尘占日产出 ~60%（核心时间卡点）
- 离线经验占日产出 ~90%
- 同种族队伍 +25% 攻击和生命
- 等级压制：差 20 级 ×1.5，差 30 级 ×2.0（上限）
- 属性总跨度：蓝色 1 级 → 白色+5 的 240 级 = 1 → 878 倍
- 前 60 分钟奖励领取 70 次（前期节奏极密集）

## 7. 行动建议

### 7.1 技术选型

| 决策 | 推荐 | 理由 |
|------|------|------|
| **游戏引擎** | **Unity**（首选）/ Godot 4（备选） | Unity 市占率 51%，放置 RPG 适配度最高，Asset Store 资源最丰富；Godot 免费开源、学习曲线低、适合纯学习 |
| **渲染模式** | 2D 竖屏 | 放置 RPG 本质是 2D 游戏，竖屏适合手机 |
| **数据存储** | JSON/SQLite 本地存储 | MVP 阶段纯本地计算离线收益，后续再加服务端 |
| **美术资源** | AI 生成 + 免费素材包 | 先验证玩法再投入美术 |
| **架构** | 模块化（战斗/养成/抽卡/关卡分离） | 便于后续扩展 |

### 7.2 学习路径（7 步走）

| 步骤 | 内容 | 目标 |
|------|------|------|
| 1 | 最简放置战斗 demo（1 英雄自动打怪，离线获经验） | 验证引擎基础能力 |
| 2 | 多英雄 + 阵营克制 + 简单关卡 | 验证战斗策略 |
| 3 | 抽卡 + 英雄升品质 | 验证收集循环 |
| 4 | 离线收益完整逻辑 + 共鸣水晶 | 验证核心循环 |
| 5 | 任务系统 + 前期节奏编排 | 验证新手体验 |
| 6 | 基础商业化（限时礼包） | 验证付费模型 |
| 7 | 数值平衡打磨 + 前期体验优化 | 追求可玩性 |

### 7.3 MVP 新手体验设计建议（来自瑞思）

1. **精简系统至 10 个以内**，避免 AFK Arena 的 60+ 系统认知过载
2. **设计"安全网"机制**，关键资源操作提供确认/撤销
3. **共享等级系统必须做**——AFK Arena 最成功的减负设计
4. **首周密集奖励曲线**，确保每日都有明显成长反馈
5. **卡关时提供明确指引**，不是"战力不足"，而是具体建议
6. **风格统一的 2D 美术 + 优质原创音乐**——独立游戏差异化机会

### 7.4 风险提示

1. **数值平衡风险**：AFK 的数值经过大量调优，直接照搬参数可能不适配。先用简化参数跑通，再逐步逼近
2. **美术品质风险**：选择差异化题材（像素风/简笔画风），避开与 AFK 正面竞争
3. **内容量风险**：MVP 控制 10-20 英雄、50-100 关，验证核心循环后再扩
4. **合规风险**：抽卡概率需公示（中国/欧盟均有法规），保底机制需明确
